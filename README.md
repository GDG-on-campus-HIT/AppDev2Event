# App Development Workshop Repositories

This repository set contains three different versions of the same Flutter app demonstrated during the sessions. Each folder represents a different stage of UI development and structuring.

## Folder Overview

### 1. basic_structural
- This version was used during the online session.
- Focuses on basic layout understanding using simple Row and Column structures.
- No advanced responsiveness or UI enhancements.
- Intended for beginners to clearly understand the fundamentals.

### 2. the_one_with_mQ
- An improved UI version of the basic app.
- Introduces MediaQuery (mQ) for responsiveness.
- Better spacing, alignment, and screen adaptation.
- Helps students understand how to make apps work across different screen sizes.

### 3. ui_improved_structural
- This version was demonstrated in the offline session.
- Focuses on a more refined UI structure.
- Cleaner layout, better organization, and improved visual hierarchy.
- Represents a more practical and polished implementation.

## Files Included
Each folder contains:
- main.dart – Entry point of the application
- home_screen.dart – Core UI implementation

## Purpose
The goal of these repositories is to show the progression from basic to improved UI design, helping learners understand:
- How to structure Flutter apps
- How to enhance UI step by step
- How to introduce responsiveness

## How to Use
- Start with basic_structural
- Move to the_one_with_mQ
- Finally explore ui_improved_structural

This sequence will help you understand the evolution of the app clearly.
